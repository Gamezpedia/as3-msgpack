Packageorg.msgpack
Classpublic class Factory
InheritanceFactory Object

The factory class is responsible for managing the workers which will encode/decode data. Each MsgPack instance has it own factory. You shouldn't instantiate this class using operator new. Instances are created internally by MsgPack objects.
