MsgPack()Constructor
public function MsgPack(flags:uint = 0)

Create a new instance of MsgPack capable of reading/writing data. You can decode streaming data using the method read. The standard workers are:

  • NullWorker: null
  • BooleanWorker: Boolean
  • IntegerWorker: int and uint
  • NumberWorker: Number
  • ArrayWorker: Array
  • RawWorker: ByteArray or String
  • MapWorker: Object

    • Parameters
    flags:uint (default = 0) — Set of flags capable of customizing the runtime behavior of this object.

    read()method
    public function read(input:IDataInput):*

    Read an object from input buffer. This method supports streaming. If the object cannot be completely decoded (not all bytes available in input), incomplete object is returned. However, the internal state (the part that was already decoded) is saved. Thus, you can read from a stream if you make successive calls to this method. If all bytes are available, the decoded object is returned.

    Parameters

    input:IDataInput — Any object that implements IDataInput interface (ByteArray, Socket, URLStream, etc).

    Returns
    * — Return the decoded object if all bytes were available in the input stream, otherwise returns incomplete object.
